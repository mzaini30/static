# Static

Simple SSG

## Prepare

```bash
npm i -g pnpm
```

## Download & Install

```bash
pnpx degit mzaini30/static static
cd static/
pnpm i
```

## Templating

## Styling

## Developing

```bash
pnpm run dev
```

## Publish

```bash
pnpm run build
```
